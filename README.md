# Installtion 

```bash
git clone https://github.com/bnal56ab/Crypto
cd Crypto
pip install -r requirements.txt
```

# Usage

To get options
```bash
python Start.py -h
```

Example
```
python Start.py -C 3 -T "Hello" -c E
```
Output
```Text
SGVsbG8=
```
